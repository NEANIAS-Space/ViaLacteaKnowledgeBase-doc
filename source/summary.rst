Summary
=======
This is summary table showing parameters in query string. All parameters must appear at most once in the query string.
The sky coordinates are interpreted in Galactic coordinate system and velocities are referenced to LSRK frame.

.. csv-table:: VLKB parameters in URL query string (mandatory, optional or not applicable)
   :file: summary.csv
   :widths: 25, 15, 15, 15, 30
   :header-rows: 1

The mandatory pubdid parameter holds only one PublisherDID in case of cutout. It holds a list of PublisherDID's
in case of merge, separated by semicolons.
